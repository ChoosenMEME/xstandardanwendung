{% trans "Begriffe und Erklärungen" %}

{% trans "Begriff" %} {% trans "Erklärung" %}
{% trans "Gewerbesteuer" %} {% trans "Eine kommunale Steuer auf den Gewerbeertrag von Unternehmen. Wird von der Gemeinde erhoben." %}
{% trans "Gewerbesteuermessbetrag" %} {% trans "Der vom Finanzamt festgesetzte Grundbetrag. Wird mit dem Hebesatz der Gemeinde multipliziert, um die Gewerbesteuer zu berechnen." %}
{% trans "Hebesatz" %} {% trans "Ein Prozentsatz, den jede Gemeinde selbst festlegt. Wird auf den Messbetrag angewendet." %}
{% trans "Zahlbetrag" %} {% trans "Der tatsächlich zu zahlende oder zu erstattende Betrag laut Bescheid." %}
{% trans "Erhebungszeitraum" %} {% trans "Das Steuerjahr oder der Zeitraum, auf den sich der Bescheid bezieht." %}
{% trans "Vorauszahlung" %} {% trans "Eine vorläufige Zahlung auf die voraussichtliche Gewerbesteuerschuld für den laufenden oder folgenden Zeitraum." %}
{% trans "XGewerbesteuer" %} {% trans "Ein standardisiertes XML-Format für den digitalen Austausch von Gewerbesteuerbescheiden zwischen Behörden und Unternehmen." %}
